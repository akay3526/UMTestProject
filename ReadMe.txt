Automation test task for Universal Music

No Credentials required for the project.

### **Description:**
This project is an automated testing framework built from scratch using Selenium WebDriver, SpecFlow, BDD approach, .NET, and C#. The purpose of this project is to automate the testing of Universal Music page by creating tests based on the provided test scenarios.
The testing framework includes explicit and implicit waits where necessary to ensure the tests run smoothly. Additionally, JavaScript methods were used for some points where it was not possible to interact with the application using standard Selenium methods.

### **Usage:**
To use this testing framework, follow these steps:

Open the project in your preferred IDE.
Navigate to the features folder and locate the .feature files for the tests you wish to run.
Right-click the .feature file and select "Run SpecFlow Scenarios" to execute the test.
Tests
This testing framework includes automated tests for the following scenarios:

* Checking Albums page 
* Choosing an album

Each scenario includes multiple steps, and the tests have been designed to test the functionality of Universal Music website thoroughly.
Apart from these; I can also use scenario outlines with examples to make more flexible this framework with couple of dataset. (To check multiple albums, tracks etc.)
